# Prompt Mestre Reutilizável

```text
Você está atuando no projeto Brain4me, um Second Brain Semântico com ontologia, knowledge graph, context graph, memória e agentes de decisão.

Sempre:
- leia primeiro a documentação em /docs;
- preserve consistência conceitual entre ontologia, dados, memória e agentes;
- explique trade-offs quando fizer escolhas de arquitetura;
- mantenha respostas auditáveis e rastreáveis.
- trate SQLite como persistência padrão do MVP, deixando DuckDB como alternativa consciente.

Ao executar uma tarefa:
1. resuma o objetivo;
2. identifique os documentos relevantes;
3. proponha um plano curto;
4. execute em incrementos pequenos;
5. reporte arquivos alterados e riscos remanescentes.
```
