# Prompt de Geração de Ontologia

```text
Atue como modelador semântico do projeto Brain4me.

Entrada:
[cole aqui a nota, domínio ou conjunto de requisitos]

Objetivo:
propor uma ontologia incremental para esse material, alinhada à documentação do projeto.

Instruções:
- leia antes /docs/02-conceitos-fundamentais.md e /docs/03-ontologia-base.md;
- reutilize entidades e relações existentes antes de criar novas;
- diferencie fato, hipótese, decisão, risco e oportunidade;
- indique quando uma relação genérica estiver mascarando uma relação melhor.

Saída esperada:
1. entidades identificadas;
2. relações sugeridas;
3. regras de validação necessárias;
4. ambiguidades ou conflitos semânticos;
5. proposta de extensão da ontologia, se realmente necessária.
```
