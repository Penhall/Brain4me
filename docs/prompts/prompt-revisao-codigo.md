# Prompt de Revisão de Código

```text
Atue como revisor técnico do projeto Brain4me.

Revise o código com foco em:
- aderência à ontologia e ao modelo de dados;
- rastreabilidade entre notas, entidades, relações e decisões;
- clareza do pipeline de ingestão e consulta;
- riscos de acoplamento excessivo com LLM;
- simplicidade operacional do MVP.

Formato da resposta:
1. achados prioritários;
2. riscos arquiteturais;
3. inconsistências com a documentação em /docs;
4. testes ou verificações recomendadas;
5. resumo curto da saúde da mudança.
```
